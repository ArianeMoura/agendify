# Apresentação

**Título do Projeto:** Agendify

O **Agendify** é uma aplicação distribuída que integra Web, Mobile e API para simplificar o processo de reservas de espaços. A solução oferece um gerenciamento de ambientes compartilhados prático, escalável e seguro, garantindo eficiência no uso de recursos.

- **Arquitetura:** Sistema distribuído com comunicação via API RESTful.
- **Back-end:** Construído em C# .NET, garantindo tipagem forte e segurança nas regras de negócio.
- **Dados:** Persistência realizada em MongoDB, permitindo modelagem flexível de reservas e usuários.
- **Front-end Web:** Interface administrativa responsiva para gestão completa do sistema.
- **Front-end Mobile:** Aplicativo nativo/híbrido focado na agilidade do usuário final para realizar reservas on-the-go.

O resultado é uma plataforma escalável que assegura a integridade dos dados e oferece uma experiência fluida tanto para gestores quanto para usuários finais.

## Identidade Visual do Agendify

Apresentamos a seguir os elementos que sustentam a personalidade do Agendify:

### 1. Logomarca:

- Linhas suaves e orgânicas;
- O ícone principal é um calendário, representando organização, planejamento e tempo;
- O detalhe do check (✓) com círculo amarelo/alaranjado no centro do calendário simboliza a confirmação, o sucesso e a eficiência da reserva. Transmite a ideia de que a tarefa foi concluída com sucesso: "Está agendado!".

### 2. Paleta de Cores:

Utilizamos o roxo como cor principal do Agendify, com o amarelo e o laranja como cores de destaque, garantindo uma identidade visual moderna, vibrante e confiável.

- Roxo (principal): transmite modernidade, criatividade e confiança;
- Amarelo e Laranja (destaque): representam sucesso, atenção e ação;
- Branco / Cinza Claro: cores de fundo que garantem contraste, legibilidade e uma UX limpa.

### 3. Tipografia:

Font-family: **Opun Mai**

A Opun Mai é utilizada para títulos de seção e subtítulos. É uma fonte display com personalidade única, mas que mantém a clareza. Ela ajuda a criar hierarquia visual e a reforçar a identidade moderna e distinta da aplicação.

Font-family: **Oleo Script**

Escolhida para a assinatura da marca (logomarca) ou para títulos de grande destaque, a Oleo Script possui um estilo script amigável e fluido. Ela adiciona um toque humano e acessível, quebrando a seriedade da estrutura de agendamento e reforçando o valor da "simplicidade".

Font-family: **Inter**

Para textos corridos, descrições, parágrafos e conteúdos mais extensos, optamos pela Inter. Trata-se de uma fonte projetada especificamente para interfaces digitais (UI), garantindo excelente legibilidade e conforto de leitura em diferentes tamanhos de tela, sendo crucial para a usabilidade do app e do site web.
