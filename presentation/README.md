# Apresentação da Solução

O **Agendify** é um sistema de **reservas de espaços compartilhados** (salas de reunião, salões,
áreas comuns) para organizações como condomínios, coworkings e empresas. Cada organização
gerencia seus espaços pelo **painel web** e seus membros reservam pelo **aplicativo mobile**;
uma única **API REST** (.NET + PostgreSQL) atende os dois clientes e garante, no banco, que
nunca existam duas reservas confirmadas para o mesmo espaço no mesmo horário.
